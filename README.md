# 南京晓庄学院军事战争沉浸式互动平台原型

这是一个面向大学生《军事理论课》课改的静态前端原型，当前聚焦“毛泽东军事思想”板块下的“四渡赤水战役”模块。

## 已实现

- 学生注册入口与登录状态展示
- 四渡赤水战役沉浸式主视窗
- 裸眼 3D 视角模拟与军事地图层切换
- 一渡、二渡、三渡、四渡赤水时间轴
- 毛泽东第一视角的态势判断与决策推演
- 战役热点、资料依据切换、学习记录保存反馈
- 军棋推演比赛入口与小组积分展示

## 本地运行

直接用浏览器打开 `index.html` 即可运行。

如果希望用本地静态服务器预览，可以在项目目录运行：

```powershell
python -m http.server 8080
```

然后访问 `http://127.0.0.1:8080`。

## GitHub Pages 部署

1. 将本目录提交到 GitHub 仓库。
2. 在仓库设置中打开 `Settings -> Pages`。
3. Source 选择 `Deploy from a branch`。
4. Branch 选择 `main`，目录选择 `/root`。
5. 保存后等待 GitHub Pages 生成访问地址。
